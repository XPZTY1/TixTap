## 🕹️ About The Game

<b>TixTap!</b> คือเกมแนว <b>Reaction Challenge</b>  
ที่ผู้เล่นต้องอ่านคำสั่งแล้วกดปุ่มให้ถูกต้องภายในเวลาที่กำหนด!
  
และกดผิดเพียงครั้งเดียว...
